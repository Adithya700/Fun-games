# ✊✋✌️ Rock Paper Scissors Game

A simple command-line implementation of the classic **Rock-Paper-Scissors** game built using Python. The player competes against the computer in a **Best of 5** match, with scores tracked after each round.

---

##  Features

* Interactive command-line gameplay
* Random computer moves
* Best of 5 rounds
* Real-time score tracking
* Tie handling
* Emoji representation for each choice

## How to Play

### Objective

Defeat the computer by winning the most rounds out of 5.

### Rules

* ✊ Rock beats ✌️ Scissors
* ✌️ Scissors beats ✋ Paper
* ✋ Paper beats ✊ Rock
* If both players choose the same option, the round ends in a tie.

---

## Running the Game

Open a terminal in the project directory and run:

```bash
python rock_paper_scissors.py
```
