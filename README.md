#  Number Guessing Game

A fun Python-based Number Guessing Game where the player has to guess a randomly generated number between **1 and 100** within **5 attempts**. This project includes both a **Command Line Interface (CLI)** version and a **Graphical User Interface (GUI)** version built using Tkinter.

---

##  Features

###  Console Version

* Random number generation between 1 and 100
* Maximum of 5 attempts per game
* Hints indicating whether the guess is too high or too low
* Option to play multiple rounds
* Displays win or lose messages

###  GUI Version (Tkinter)

* User-friendly graphical interface
* Input field for guesses
* Real-time feedback messages
* Displays remaining attempts
* Play Again button
* Exit button
* Message boxes for winning and losing scenarios
* Input validation for non-integer values

##  How to Play

### Objective

Guess the secret number generated by the computer between **1 and 100**.

### Rules

* You have only **5 attempts**.
* After each guess, you'll receive a hint:

  * 📉 Too low! Try again.
  * 📈 Too high! Try again.
* Guess the correct number within 5 attempts to win.
* If you fail, the correct number is revealed.
